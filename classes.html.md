# 訓練課程

和風健身中心每週提供超過 50 堂團體課程，涵蓋拳擊有氧、瑜伽、劍道體能、空手道、動感單車及普拉提，並提供一對一私人訓練。

## 團體課程

| 課程 | 時長 | 強度 |
|------|------|------|
| 和風拳擊有氧 | 60 分鐘 | 高 |
| 禪意瑜伽 | 75 分鐘 | 低 |
| 劍道體能訓練 | 50 分鐘 | 中 |
| 空手道健身 | 55 分鐘 | 中 |
| 動感單車 | 45 分鐘 | 高 |
| 普拉提 | 50 分鐘 | 低 |

包含體能評估、目標導向訓練計劃及進度追蹤。
